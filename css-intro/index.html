CSS coding styles

Browser Defaults You have not supplied any css code
Inline This is for a single specific element (tag) instance. The element contains a parameter called style='' style="rule; [rule;...]" style="color:red; width:50px;"
Embedded Coded in your head area of the web page. Is started with the style element and contains the selctors and rules for the desired CSS. Applies only to the physical page where the code is placed. Effects all selectors on the page. <style>
selector {rule; [rule;....]}
[selector {rule; [rule;....]}...]
<style>		 <style>
p {font-size:large; font-weight:bold;},
{width:350px;}
</style>
Style Sheet This is similar to embedded, however the selectors and rules are coded in a seperate physical file. This file can be shared throughout the entire website. Thus all CSS is in one common place. p {font-size:large; font-weight:bold;},
[blockquote {width:350px;}
